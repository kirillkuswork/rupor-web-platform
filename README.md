# rupor-web-platform
- Видеоплатформа на базе рутуб. Можно смотреть видео, писать комментарии, ставить лайки, добавлять в избранное, смотреть историю и т.п, по функционалу похоже на Rutube или Youtube.
- Проект на React, Typescript, NextJS.
- Архитектура FSD.
- Есть отдельный ui-kit и отдельный проект для работы с переводом.
